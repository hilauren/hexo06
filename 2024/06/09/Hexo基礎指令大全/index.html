一起來練習如何用Hexo架網站

首頁 部落格 關於我
2024-06-09

Hexo基礎指令大全

  1. 建立部落格：hexo init "你的網站名稱"
  2. 開啟伺服器（產出的網址可點擊預覽網站）：hexo server
  3. 新增一篇部落格：hexo new "部落格標題"
  4. 輸出實際網頁： hexo generate
  5. 清除快取：hexo clean
  6. 部署網站：hexo deploy
  7. 部署網站：hexo new page 頁面名稱
